To get Ubuntu 16.04 to boot to `non-graphics mode` the following steps worked for me:

1. Change the following lines in the `/etc/default/grub` file:

        GRUB_CMDLINE_LINUX_DEFAULT="text"
        GRUB_CMDLINE_LINUX="text"
2. Then set the default runlevel with `systemd`:

        sudo systemctl set-default multi-user.target
3. Reboot

4. To return to normal `graphic mode`:

        sudo systemctl set-default graphical.target
